# Node.js Version Information

This project uses Node.js v22.21.0 with npm v10.9.4.

## Requirements
- Minimum Node.js version: 18.17.0
- Recommended Node.js version: 20.x (LTS) or 22.x (Current)

## Version Compatibility
| Component | Version | Status |
|-----------|---------|---------|
| Node.js | v22.21.0 | ✅ Compatible |
| npm | v10.9.4 | ✅ Compatible |
| Next.js | 15.3.5 | ✅ Compatible |
| React | 19.0.0 | ✅ Compatible |

## Installation
1. Install Node.js 18.17.0 or higher
2. Install project dependencies:
   ```bash
   npm install
   ```

## Development
```bash
npm run dev
```

## Production
```bash
npm run build
npm start
```
